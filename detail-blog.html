Belajar HTML Dasar untuk Pemula

Bunga Rhiza Sitorus | 23 Agustus 2025

HTML Web Frontend

Deskripsi

Artikel ini menjelaskan konsep dasar HTML: struktur dokumen, elemen-elemen penting seperti <head>, <body>, heading, paragraf, tautan, gambar, dan form.

Pendahuluan

HTML (HyperText Markup Language) adalah bahasa markup dasar untuk membuat halaman web. Semua elemen website seperti teks, gambar, tautan, dan form dibuat menggunakan HTML.

1. Struktur Dasar HTML
  • Setiap dokumen HTML diawali dengan <!DOCTYPE html>.
  • Elemen utama: <html>, <head>, <body>.
  • Bagian <head> berisi meta, title, link CSS, dan script.
2. Elemen Penting
  • Heading: <h1> - <h6>
  • Paragraf: <p>
  • Tautan: <a href="url">
  • Gambar: <img src="..." alt="...">
  • Form: <form>, <input>, <textarea>
3. Contoh Implementasi
          <!DOCTYPE html>
          <html>
          <head>
            <title>Halaman Pertama</title>
          </head>
          <body>
            <h1>Halo Dunia!</h1>
            <p>Ini paragraf pertama saya.</p>
            <a href="https://example.com">Kunjungi contoh</a>
          </body>
          </html>
4. Tips & Trik
  • Gunakan komentar <!-- Komentar --> untuk catatan.
  • Selalu tutup tag yang diperlukan.
  • Gunakan struktur rapi agar mudah dibaca.
